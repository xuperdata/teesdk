## paillier 
A Paillier SDK by Golang as a wrapper of libpaillier(http://hms.isi.jhu.edu/acsc/libpaillier/)

## Usage 

1. run the unit test  
```
go test
```
or your program.


2. run benchmark test
```
go test -bench=.
```
